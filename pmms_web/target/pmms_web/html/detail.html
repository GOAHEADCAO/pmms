园艺学院开展“2018届本科毕业论文撰写规范培训会”

作者：水水水 发布日期：2017-06-09

9月22日下午，在成都校区二教报告厅，借全院毕业生大会召开之际，学院为2018届全体本科生进行了“本 科毕业论文撰写规范培训会”。 学院按照教务处对“2014级本科毕业论文（设计）和毕业实习工作”、“关于2018届毕业论文开题报告检查”等工作的安排部署，结合近几年在本科毕业论文收集、整理、检查等工作中的经验、出现问题较多的地方，通过培训会这一方式，向全体同学进行规范和说明，旨在提高全院论文质量，迎接本科教学审核评估。 培训会首先由教学院长对全体毕业生进行动员，指出本科毕业论文是本科生的重要实践教学环节，在本科评估检查工作中的重要位置，要求全体毕业生要高度重视这一环节，保质保量的完成毕业论文试验和毕业论文撰写工作。学院教学秘书以制作的论文范本为例，通过PPT将论文各个部分的填写要求，需要注意的细节向全体毕业生做规范和说明，对同学疑问较多的地方做出解答。此外，学院在论文管理工作上通过抓好两个源头一个中间环节，即对毕业生和答辩组秘书两个源头分类别做好论文撰写和论文收集的培训，以指导教师作为最重要中间环节，答辩小组和院督导严把质量关，共促本科毕业论文质量提升。
